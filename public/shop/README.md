# Imagens estáticas da loja

Coloque aqui as imagens dos produtos da Loja.

No GitHub Pages, tudo dentro de `public/` é publicado como arquivo estático. O código usa `import.meta.env.BASE_URL`, então os caminhos funcionam tanto em desenvolvimento quanto em `/sapataria-digital/`.

Você pode substituir os arquivos de exemplo mantendo os nomes atuais, ou alterar os nomes em `src/pages/Shop.tsx`.
